## Технологии
- **HTML5** - семантическая разметка
- **CSS3** - Flexbox, Grid, анимации, медиазапросы
- **JavaScript** - DOM манипуляции, обработка событий
- **Font Awesome** - иконки
- **Google Fonts** - типографика

## Цветовая палитра
- Основной: #8a9a5b (Арктическая ива)
- Акцентный: #6b8e23 (Оливковый)
- Фон: #2c3e20 (Тёмно-зелёный)
- Текст: #333333 (Тёмно-серый)

## Как использовать
1. Клонируй репозиторий
2. Замени `photo.jpg` на своё фото
3. Отредактируй данные в `index.html`
4. Открой `index.html` в браузере

## Деплой
Сайт автоматически деплоится через GitHub Pages.
